淘系推广参谋 · 油猴脚本使用帮助

如果你点启动器的「打开 Tampermonkey 管理面板」后看到 ERR_BLOCKED_BY_CLIENT，是因为 Chrome 安全策略禁止外部程序直接打开油猴内部页面，这是正常的。

1先确认 Tampermonkey 是不是正版

注意：我之前的说明把正版 ID 写错了，非常抱歉。请以官方 FAQ 为准。

打开 Chrome 的「扩展程序」页（地址栏输入 chrome://extensions/），找到 Tampermonkey/篡改猴，查看它的 ID：

  • 正版 Chrome ID：dhdgffkkebhmkfjojejmpbldmpobfkfo（注意中间是 ejmp）
  • 正版 Edge ID：iikmkjmpaadaobahmlepeloendndfphd

如果你看到的是其他 ID，请点击该扩展右下角的「移除」按钮卸载，然后回到启动器点「安装 Tampermonkey 扩展（正版）」重新安装。官方参考：Tampermonkey 官方 FAQ - 版本与 ID

2脚本到底装好了没有？

装好了。

只要你在 Tampermonkey 安装页看到类似「即将重新安装用户脚本 v2.1」或「安装」按钮，就说明脚本已经在 Tampermonkey 里了。

油猴用户脚本不会出现在 Chrome 的「扩展程序」页里，那里只能看到 Tampermonkey 扩展本身。你写的脚本要在油猴自己的「管理面板」里看。

3怎么打开 Tampermonkey 管理面板？

方法很简单：在 Chrome 浏览器右上角找到 Tampermonkey（油猴）图标，点击它，然后选择「管理面板」/「Dashboard」。

如果右上角没有图标，说明它被折叠到「拼图」图标里了，点一下拼图图标，找到 Tampermonkey 旁边的图钉，把它固定到工具栏。

4管理面板里应该看到什么？

你应该能看到一条脚本：

淘系推广参谋 · 一键导入（悬浮按钮）

状态是「已启用」，包含 URL：

  • https://*taobao.com/*
  • https://*tmall.com/*
  • http://127.0.0.1:8123/*

如果看到这条，就说明脚本已经生效。

5怎么在商品页使用？

  1. 打开任意淘宝或天猫商品详情页（例如 item.taobao.com 或 detail.tmall.com）。
  2. 页面加载完成后，右下角会出现一个橙色的悬浮按钮：🛒 导入推广参谋。
  3. 点击按钮，脚本会自动抓取标题、价格、主图，并打开本地工具页。
如果没看到悬浮按钮，请刷新页面，或检查 Tampermonkey 图标上是否有数字/红底提示，点进去启用脚本。

6还是不行？

回到本工具页 http://127.0.0.1:8123/，在「商品链接」框上方点 「复制脚本代码（手动安装）」，然后：

  1. 点击 Tampermonkey 图标 → 管理面板。
  2. 点左侧「+ 添加新脚本」。
  3. 删除编辑器里的默认内容，粘贴刚才复制的代码。
  4. 按 Ctrl + S 保存。
