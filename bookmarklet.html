淘系推广参谋 · 浏览器抓取书签

不用申请淘宝官方 API，利用你当前浏览器登录态，一键把商品信息导入本地工具。

⚠ 请使用 Chrome 或 Edge

360 浏览器会导致本地工具页面卡加载。 推荐使用 Google Chrome 或 Microsoft Edge。

如果你还没安装 Chrome，请先安装；本工具的一键启动器会自动调用 Chrome。

推荐方式：在工具页一键安装

双击 运行工具.bat → 点 ① 启动并打开工具页 → 在页面顶部点「安装」按钮，即可把书签自动写入浏览器书签栏。

备用方式：把下面这个按钮拖到浏览器书签栏

按住鼠标左键，把橙色按钮拖到你的浏览器书签栏（Chrome/Edge 地址栏下方），松开后即可。

注意事项

  • 请使用 Google Chrome 或 Microsoft Edge 打开本工具。 360 浏览器会导致本地页面卡加载。
  • 使用前请确保本地工具已启动（双击 运行工具.bat）。
  • 书签只能抓取当前浏览器里已经打开的商品页；因为用的是你的登录态，所以不会被淘宝验证页拦截。
  • 如果价格没抓出来，请确认页面已加载完成再点书签。
  • 如果书签栏没有显示，按 Ctrl+Shift+B（Windows）打开。
